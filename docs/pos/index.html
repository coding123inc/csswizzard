Positions

The following HTML Classes define CSS Positions

Set fixed position of element

CSS Equivalent => position: fixed;

<div class="fixed">Fixed</div>

Set absolute position of element

CSS Equivalent => position: absolute;

<div class="absolute">Fixed</div>

Set relative position of element

CSS Equivalent => position: relative;

<div class="relative">Fixed</div>

Display

The following HTML Classes define CSS Display

Set inline display of element

CSS Equivalent => display: inline;

<div class="inline">Fixed</div>

Defines a flex element

CSS Equivalent => display: flex;

<div class="flex">Fixed</div>

Hides an element

CSS Equivalent => display: none;

<div class="noDisplay">Fixed</div>

Hides an element in mobile devices

CSS Equivalent

@media screen and(max-width:980px) {

.notShownOnMobile {

display:none;

}

}

<div class="notShownOnMobile">Fixed</div>

Show an element

CSS Equivalent => display: block;

<div class="block">Fixed</div>
