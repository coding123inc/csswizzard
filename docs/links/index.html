Delete underline & color

Smooth scroll

Use the sm-sc class to create a smooth scroll

Scroll Top

<a href="body" class="sm-sc noUnderline">Scroll Top</a>

The data-to attribute can be used instead of href on any element

Scroll Top

<a data-to="body" class="sm-sc noUnderline">Scroll Top</a>

HREF

The element that window will be scrolled to needs to be a query selector or CSS selector

Scrolling to tag: data-to="tagName"

Scrolling to specific HTML Class: data-to=".className"

Scrolling to specific id: data-to="#idName"
