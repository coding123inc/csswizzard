  • .noWeight - 400 Weight

  • .fw100 - 100 Weight

  • .fw200 - 200 Weight

  • .fw300 - 300 Weight

  • .fw400 - 400 Weight

  • .fw500 - 500 Weight

  • .fw600 - 600 Weight

  • .fw700 - 700 Weight

  • .fw800 - 800 Weight

  • .fw900 - 900 Weight

  • .bold - Bold Weight

  • .bolder - Bolder Weight

  • .wl - Lighter Weight

  • .winh - Inherit Weight

  • .wini - Initial Weight

  • .wun - Unset Weight
