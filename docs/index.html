Installation

Copy and paste the following tag into your <head> tag

<link rel="stylesheet" type="text/css" href="https://csswizzard.cf/code/min.css" />

Now, CSSWizzard is available in NPM

npm install csswizz

And in JsDeliver

<link rel="stylesheet" crossorigin="anonymous" href="https://cdn.jsdelivr.net/npm/csswizz@1.1.0/src/min.css" integrity="sha256-ywOG2sH0bOQ99/qCYINbWR+fK29M7yEEFBJGVoMkOTY=" />

Responsive HTML Tag

Insert this tag in your website's <head> tag

<meta name="viewport" content="width=device-width, initial-scale=1" />
