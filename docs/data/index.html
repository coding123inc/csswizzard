Area scale

With the data-scale attribute, the area size can be controlled:

It works with select, input and textarea tags

Also, works with p and span

Textarea properties

  • width: 95%

  • height: 300px

Input properties

  • width: 95%

Select properties

  • width: 95%

<input data-scale="large">

Textarea properties

  • width: 50%

  • height: 50%

Input properties

  • width: 50%

Select properties

  • width: 50%

<input data-scale="middle">

Textarea, Input, Select properties

  • width: 35%

<input data-scale="tiny">

Only allowed in p and span tags

Hello World

<p data-scale="title">Text</p>

ALTERNATIVE: If need to set a title without p or span tag, use the title class

<p class="title">Text</p>

Only allowed in p and span tags

Hello World

<p data-scale="subtitle">Subtitle</p>

ALTERNATIVE: If need to set a title without p or span tag, use the subtitle class

<p class="subtitle">Text</p>

Textarea resizing

The data-resize attribute define a textarea box resizing

Disables vertical & horizontal resizing of area

<textarea data-resize="no">Text</textarea>

Enables vertical-only resizing of area

<textarea data-resize="vertical">Text</textarea>

Enables horizontal-only resizing of area

<textarea data-resize="horizontal">Text</textarea>

Element Filter

The data-filter attribute set element's filter. This attribute is supported on any element

Equivalent: [Value of Attribute]px. Ex: data-filter="blur 30" -> blur(30px)

Starting at 10

<div data-filter="10 blur">data-filter</div>

data-filter

Ending at 650

<div data-filter="650 blur">data-filter</div>

Equivalent: [Value of Attribute]px. Ex: data-filter="grey 30" -> grayscale(30%)

Starting at 1

<div data-filter="1 grey">data-filter</div>

Ending at 100

<div data-filter="100 grey">data-filter</div>

Equivalent: [Value of Attribute]px. Ex: data-filter="bright 30" -> brightness(30%)

Starting at 1

<div data-filter="1 bright">data-filter</div>

Ending at 200

<div data-filter="200 bright">data-filter</div>

Equivalent: [Value of Attribute]px. Ex: data-filter="bright 30" -> contrast(30%)

Starting at 1

<div data-filter="1 contrast">data-filter</div>

Ending at 200

<div data-filter="200 contrast">data-filter</div>
