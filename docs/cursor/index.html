Cursors

The following HTML Classes, define the CSS cursor property:

  • pointer

  • alias

  • all-scroll

  • cauto - Automatic cursor

  • c-cell - Cell cursor

  • col-resize

  • c-copy - Copy cursor

  • crosshair

  • c-d - Default cursor

  • ew-resize

  • e-resize

  • grab

  • grabbing

  • c-help - Help cursor

  • c-move - Move cursor

  • c-vr - Vertical resize cursor

  • c-nr - Oblique resize cursor

  • c-nr - No drop cursor

  • c-inv - No cursor

  • c-na - Not allowed cursor

  • c-p - Progress cursor

  • c-r - Row Resize cursor

  • c-sr - Vertical Resize cursor

  • c-r - Text cursor

  • c-vt - Vertical Text cursor

  • c-wait - Wait cursor

  • c-zi - Zoom In cursor

  • c-zo - Zoom Out cursor
