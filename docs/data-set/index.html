Textarea data-resize

With the data-resize, the textarea element can have a horizontal, vertical or no resize

<textarea data-resize="no"></textarea>
<textarea data-resize="horizontal"></textarea>
<textarea data-resize="vertical"></textarea>


data-bord

With data-bord, the element will have a custom border. Any element can have the data-bord attribute

Allowed values:

  • blue

  • red

  • green

  • orange

  • purple

  • pink

Purple border
<div class="container" data-bord="purple">Purple border</div>
Orange border
<div class="container" data-bord="orange">Orange border</div>
Green border
<div class="container" data-bord="green">Green border</div>
Red border
<div class="container" data-bord="red">Red border</div>
Blue border
<div class="container" data-bord="blue">Blue border</div>
Pink border
<div class="container" data-bord="pink">Pink border</div>


data-align

The data-align attribute modifies the text align of element. Any element can have the data-align attribute

Allowed values:

  • right

  • left

  • center

Centered text
<div class="container" data-align="center">Centered Text</div>
Text align in left side
<div class="container" data-align="left">Text align in left side</div>
Text align in right side
<div class="container" data-align="right">Text align in right side</div>


data-b

The data-b attribute modifies the background color of element. Any element can have the data-b attribute

Allowed values:

  • 12

  • 11

  • 10

  • 0

  • c2

  • c1

  • c

  • f1

  • f2

  • f3

  • f4

  • f5

  • f6

  • f7

  • f8

  • f9

  • f

Quick Reminder

The value of data-b will be converted to HEX: #(Value)

If the value is c, the background of element will be #ccc

If the value is f2, the background of element will be #f2f2f2
